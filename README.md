# 🍽️ Lunch Sync

A simple web app for two people working from home to coordinate lunch. Drag to set your available window, see your partner's in real time, and find when you can eat together.

**Live at:** https://williamdxz.github.io/lunch-sync/

## Features

- **Drag-to-select grid** — set your lunch window in 15-minute increments
- **Real-time sync** — updates appear instantly on both devices via Supabase
- **Overlap detection** — see exactly when you can eat together
- **Mobile-first** — designed for quick use on your phone during the workday
- **Resets daily** — only shows today's availability
- **No accounts** — just pick Person 1 or Person 2 on each device

## Tech

Single HTML file hosted on GitHub Pages, backed by Supabase Realtime Database (free tier). No build step, no dependencies beyond the Supabase CDN.
